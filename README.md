# dxrmb

[![npm](https://img.shields.io/npm/v/dxrmb)](https://www.npmjs.com/package/dxrmb)

大写人民币

## 安装

```sh
npm i dxrmb
```

## 导入
```javascript
const dxrmb = require("dxrmb");
```
或者
```javascript
import dxrmb from "dxrmb";
```

## 使用
```javascript
const rmb = dxrmb('456789.23');
console.log(rmb)
// 肆拾伍万陆仟柒佰捌拾玖元贰角叁分
```
